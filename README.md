# Money Manager (scaffold)

This workspace contains a minimal scaffold of a React Native (Expo) app based on the provided technical specification `README_Money_Manager.md`.

Next steps:

- Run `npm install` or `yarn` to install dependencies.
- Run `npm start` to open Expo.

Files added:
- `App.js` - navigation and DB init
- `src/database/init.js` - creates SQLite schema
- `src/services/transactions.js` - basic transaction service
- Basic screens under `src/screens`
